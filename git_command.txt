git clone "Ссылка на репозиторий" - клонирует репозиторий в текущую папку и задает имя папки именем репозитория.
git clone "Ссылка на репозиторий" . -  клонирует репозиторий в текущую папку
git clone "Ссылка на репозиторий" "Имя папки" - клонирует репозиторий в текущую папку с заданным именем

git status - показывает состояние репозитория

git add "File name" - добавить файл в рабочее дерево репозитория
git add . - добавить в дерево все изменения всех файлов

git commit "комментарий" - подтвердить изменения
